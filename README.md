# zepto-SQL-data-analysis

This project analyzes an e-commerce inventory dataset from Zepto, a quick-commerce platform known for fast grocery delivery in India. The goal of this project is to simulate a real-world Data Analyst workflow using SQL for data exploration, cleaning, and business insights. ()

📂 Dataset

The dataset contains product-level inventory information including category, price, discount, stock availability, and quantity.

Main columns used:

Category

Product Name

MRP

Discount Percentage

Available Quantity

Discounted Selling Price

Weight (grams)

Stock Status

Quantity Sold

🛠 Tools Used

PostgreSQL

pgAdmin

SQL

🔧 Project Steps

Created a database and table schema in PostgreSQL

Imported the dataset using pgAdmin

Performed data exploration to understand the dataset

Cleaned the data by handling missing and incorrect values

Wrote SQL queries to analyze product pricing, discounts, and inventory

📊 Key Analysis

Identified products with the highest discounts

Analyzed out-of-stock products

Compared product prices across categories

Estimated potential revenue based on price and quantity

Found categories with the highest average discounts

🚀 Conclusion

This project demonstrates how SQL can be used to explore, clean, and analyze real-world e-commerce data to generate meaningful business insights.
